# Cuadre

Cuadre es una base comercial para negocios que necesitan controlar ventas, inventario
y reportes desde una herramienta simple, adaptable y lista para crecer con cada cliente.

El proyecto nace a partir de la experiencia construyendo IsaCream, pero apunta a ser una
solucion mas generica: una plataforma que pueda ajustarse a distintos tipos de negocios
sin partir desde cero en cada implementacion.

## Vision

- Gestion de inventario, productos y movimientos.
- Registro de ventas con reportes operativos.
- Experiencia clara para equipos pequenos y medianos.
- Base adaptable para planes comerciales, configuraciones por cliente e integraciones.
- Mejoras progresivas para uso en moviles y flujos de venta rapidos.

## Proximos frentes

- Planes y limites por tipo de cliente.
- Ajustes de interfaz para dispositivos moviles.
- Integraciones con servicios externos.
- Configuracion flexible para nuevas verticales de negocio.
- Documentacion de trabajo para escalar el desarrollo de forma controlada.

## Desarrollo

Este proyecto usa pnpm como gestor de paquetes.

```bash
pnpm install
pnpm run dev
pnpm run build
pnpm run type-check
pnpm run lint
```
